Setting the Team Website Up

Wed, Feb 16, 2022 The Rapture ROV team website is built using Hugo static site generator with Hyde theme. In this blog I will be explaining the why’s and how’s of this decision. Why Hugo? Hugo is a static site generator written in Go programming language which is well known for its speed and efficiency. Hugo is mostly used for serving content that doesn’t get updated often and is especially well suited for blogs. With the wide variety of community themes, Hugo can be used to build and configure a website blazing fast.
